# High Vibe Life Coach

Calm daily mindset app

## About
High Vibe Life Coach is a lightweight app that delivers short daily mindset prompts and practices to help users cultivate calm, clarity, and positive habits.

## Features
- Daily mindset cards
- Simple, calming UI
- Short guided exercises and reminders

## Development
To run locally:
1. Clone the repo
2. Install dependencies (example):
   ```bash
   npm install
   ```
3. Start the development server (example):
   ```bash
   npm run dev
   ```

(Adjust commands above to match the project's actual stack.)
